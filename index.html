Sprint 2

Je hebt vorige week je eerste stappen in het programmeren in Javascript met een webpagina geleerd. Deze week gaan we weer wat verder. De oefenopdrachten maak je tijdens de les.

Wat gaan we leren?

Tijdens de les gaan jullie oefenen met een aantal onderdelen.

  • Je weet wat de DOM is en hoe deze in elkaar zit
  • Gaan we kijken hoe we door de DOM kunnen 'lopen'
  • Leren we je om HTML-elementen in de DOM te selecteren
  • Leren we hoe je een HTML-element kan aanpassen door de style of de class aan te passen
  • En starten we met informatie over browser events
  • Uiteraard vertrouwen we erop dat je de oefen- en eindopdracht van sprint 1 gemaakt hebt

DOM? Je bent zelf dom!

Nee natuurlijk ben je niet dom, anders zou je hier niet zitten :-D

DOM staat voor Document Object Model. En in de DOM zit alles van je webpagina. Van de <!DOCTYPE html> tot en met de laatste </html> tag. Dit heb je gelezen in de paragraaf 1.1 van part 2 op de Javascript Info website

Opdracht 1

Je hebt voor deze les al de paragrafen gelezen van part 2 op de JS Info website. Neem deze nog eens even kort door.

Opdracht 2

bekijk de DOM van dit document eens. Inspecteer (gebruik je developers console) deze pagina eens, en bekijk de DOM van deze pagina.

Indenting

Indenting was natuurlijk al heel belangrijk (gebruik maken van spaties om je code in overzichtelijke stukken te hakken) het helpt developers zelf om de code goed te lezen, maar ook voor het debuggen/ zoeken naar onderdelen is dit belangrijk. Je Browser zal je helpen, maar blijf goed op je indenting letten!
