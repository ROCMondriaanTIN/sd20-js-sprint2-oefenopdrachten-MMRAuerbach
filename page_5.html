Eerste browser event

In hoofdstuk 2.1 (van part 2) heb je kunnen lezen over events in de browser (browser events). In het voorbeeld heb je gelezen dat je een bepaalde handeling/ evenement kan afvangen in de browser. In het eerste voorbeeld zie je dat dit met een HTML-attribute onclick gebeurd.

Net als het werken met ID's niet meer van deze tijd is, zijn onclicks eigenlijk ook niet meer van deze tijd. Later in de lessen serie gaan we in hoe we dit beter kunnen doen. In plaats daarvan werken we met de addEventListener en class-names. Hoe je elementen kan selecteren op class-name heb je eerder al geleerd. Hieronder een klein voorbeeldje: 

                    
                        //Eerst maken we een variabele aan (die niet kan veranderen) met het door mij gewenste element.
                        const myElement = document.querySelector('.my-class');
                        
                        //Controleer of het element bestaat
                        if (myElement) { 
                            //Voeg een eventhandler toe (addEventListener) die reageert op een click.
                            myElement.addEventListener('click', function() {
                                alert('Hello world');
                            })
                        }

Opdracht 1

Hieronder staat een button, zorg ervoor dat als je op de button klikt er een popup verschijnt die jouw naam vraagt. Vervolgens toon je de gekozen naam in de console.log.

Je hebt hier een aantal van je tot nu toe geleerde skills voor nodig, om je op weg te helpen:

  • Maken van een variabele
  • Opvangen van de waarde van de prompt popupbox in de variabele
  • Zoeken naar het door jouw gekozen element via een query-selector
  • Aanmaken van een event handler op een element
  • Afvangen van de click op het element
