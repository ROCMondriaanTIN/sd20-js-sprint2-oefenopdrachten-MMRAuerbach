Door de DOM lopen

Zoals je hebt kunnen lezen in paragraaf 1.3 van de JS Info website kan via nodes door de DOM (=webpagina) heen lopen. Hiervoor gebruik je NODES. Een NODE is niets anders dan een element op je pagina.

Introductie

Je gaat deze pagina inspecteren en leren hoe je door de DOM kan lopen.

Kijk eens in je console

Inspecteer deze pagina eens, en kijk eens in je console, je ziet daar code staan, dit ziet er ongeveer zo uit. ➤#text ➤h2 ➤#text ➤p ➤#text Probeer eens te vinden waar dit vandaan komt.

Opdracht 1

Er zijn nog twee idents op deze pagina, maak de code eens om te zorgen dat je ook daar door heen loopt.

Opdracht 2

Maak ook eens code dat je door de hele body en de head sectie van de pagina heen loopt.
