Veranderen van style

Je hebt al weer een hoop nieuwe dingen geleerd, belangrijk is om te blijven herhalen, dit zal ook in de sprint eindopdrachten terugkomen. Oefenen ook zelf, maak je Portfolio ook leuker met Javascript!

We gaan nu de style van een element veranderen van elementen die je geselecteerd hebt.

Veranderen/ toevoegen van style via het style-attribute

Zoals je hebt kunnen lezen kan je de style van een enkel element makkelijk veranderen. 

                    
                        let myElement = document.querySelector('.first-section');
                        if (myElement) { //altijd controleren of je variabele niet leeg is
                            myElement.style.backgroundColor = "red";
                        }

Het veranderen van een class-name in een element

Een betere/ voorkeurs oplossing is om een style aan te passen via een class-name. Onderstaande code voegt de class-name bg_red toe aan het class="" van het gekozen element. 

                    
                        let myElement = document.querySelector('.first-section');
                        if (myElement) { //altijd controleren of je variabele niet leeg is
                            myElement.classlist.add('bg_red');
                        }

Opdracht 1

  1. Maak een variabele die aangeeft wat een achtergrond kleur kan zijn. (bg-green)
    Zet de class ook in de CSS file main.css met een correcte waarde
  2. Maak ook een variabele die aangeeft wat een font kleur kan zijn.
    Zet de class ook in de CSS file main.css met een correcte waarde

Opdracht 2

Zoek het element first-section, en verander de achtergrond kleur naar de waarde van de in opdracht 1 opgegeven class-name.

Opdracht 3

Zoek alle elementen met de class-name normal-text. Verander van alle elementen de font-kleur, naar de kleur van de in opdracht 1 gegeven class-name.
